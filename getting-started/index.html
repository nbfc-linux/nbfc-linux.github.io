Getting Started

When running NBFC-Linux for the first time, you need to select a configuration file that matches your laptop model.
Hereâ€™s how to get started:

Update available configurations

Run the following command to download the latest configuration files: 

sudo nbfc update

Automatically select a matching configuration

Try this command to automatically detect and set a suitable configuration: 

sudo nbfc config -s auto

Find recommended configurations

If auto-detection fails, you can get a list of configurations with a similar name: 

nbfc config -r

Set a configuration

To set a configuration for a model, use: 

sudo nbfc config -s <MODEL>

Start the service

Once a configuration is set, you can start the service: 

sudo nbfc start

Query the service

You can query the service using: 

nbfc status

Start the service automatically at boot

To enable NBFC to start with the system: 

sudo systemctl enable nbfc_service
